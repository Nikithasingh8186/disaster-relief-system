# Contributing Guide

## Branch Naming

feature/<feature-name>

bugfix/<bug-name>

docs/<document-name>

## Commit Convention

feat:
fix:
docs:
refactor:
test:
chore:

Example:

feat: add audio upload endpoint

## Pull Requests

1. Create feature branch.
2. Commit changes.
3. Push branch.
4. Open merge request.
5. Request review.

## Coding Standards

### Python

* Black
* Flake8
* MyPy
* isort

### JavaScript

* ESLint
* Prettier
