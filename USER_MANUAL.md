# User Manual

## Starting Backend

uvicorn backend.app.main:app --reload

## Starting Frontend

npm run dev

## Upload Audio

1. Open Upload page.
2. Select audio file.
3. Click Submit.

## Upload Image

1. Select image.
2. Click Submit.

## Search

Search by:

- Location
- Priority
- Disaster Type

## Export

Click Export CSV.

## Offline Usage

Disconnect internet and continue using the application normally.
