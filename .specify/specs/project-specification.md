# Project Specification

## Project Name

RescueSync AI

## Objective

Convert unstructured disaster information into structured data.

---

# Functional Requirements

FR-1:
Upload audio files.

FR-2:
Upload images.

FR-3:
Submit text reports.

FR-4:
Extract structured information.

FR-5:
Store incidents.

FR-6:
Search incidents.

FR-7:
Export reports.

---

# Non Functional Requirements

NFR-1:
Works offline.

NFR-2:
CPU only.

NFR-3:
Response time below 10 seconds.

NFR-4:
SQLite storage.

NFR-5:
No cloud dependencies.
