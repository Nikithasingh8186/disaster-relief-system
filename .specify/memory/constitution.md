# RescueSync AI Constitution

Version: 1.0

## Mission

Build an offline-first, CPU-powered disaster relief intelligence system that transforms unstructured reports into structured actionable data.

---

## Core Principles

### 1. Offline First
The application must work without internet connectivity.

### 2. CPU First
All AI inference shall run on CPU only.

### 3. Privacy by Design
No user data shall leave the device.

### 4. Open Source
The entire project must be released under GPL-3.0.

### 5. Reproducibility
Every feature and experiment should be reproducible.

### 6. Reliability
The application should gracefully handle failures.

### 7. Accessibility
The system should be usable on low-resource machines.

### 8. Transparency
AI decisions should be explainable and auditable.
